# Common Linux PrivEsc – TryHackMe Writeup

This repository contains my walkthrough for the Common Linux Privilege Escalation room on TryHackMe.

The room demonstrates several common Linux privilege escalation techniques used during penetration testing.

## Skills Practiced

- Linux enumeration
- Identifying SUID binaries
- Exploiting writable system files
- Cron job exploitation
- PATH variable manipulation

## Tools Used

- LinEnum
- Linux enumeration commands
- SSH
- Netcat

## Techniques Covered

- SUID exploitation
- Writable /etc/passwd privilege escalation
- Cron job exploitation
- PATH hijacking

## Objective

Gain root privileges on the target system by identifying and exploiting misconfigurations.
